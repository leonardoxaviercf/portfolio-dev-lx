# 🚀 Portfolio Profissional V2 - Leonardo Xavier (React Edition)

Esta é a **segunda versão** do meu portfólio pessoal. O projeto representa uma evolução significativa da versão anterior, desenvolvida originalmente em HTML/CSS estático. Agora, utilizo o ecossistema **React** para oferecer uma experiência mais dinâmica, performática e modular.

O foco permanece na apresentação da minha trajetória, com o objetivo de seguir a carreira de **Desenvolvimento Backend**, agora demonstrando habilidades avançadas de Frontend moderno.

> **Status do Projeto:** 🟢 Concluído e Evoluindo  
> **Tecnologia Principal:** React + Vite

---

## 🔄 Evolução do Projeto (V1 ➡ V2)

Diferente da primeira versão, este portfólio foi totalmente reconstruído para aplicar conceitos modernos de desenvolvimento:
* **Componentização:** Interface dividida em componentes reutilizáveis como Navbar, Hero, Skills e Projects.
* **Gerenciamento de Estado:** Uso de `useState` para criar uma seção de tecnologias interativa com sistema de abas.
* **Renderização Condicional:** Lógica inteligente para exibir botões de projetos apenas quando há links de demonstração disponíveis.
* **Interatividade com Ícones:** Integração da biblioteca `react-icons` para uma interface visualmente rica.
